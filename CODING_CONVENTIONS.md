# 코딩 컨벤션
- **컴포넌트 네이밍**: PascalCase
- **파일 구조**: `src/app`, `src/components/{layout,ui,sections}`, `src/lib`
- **반응형 우선**: 모바일 퍼스트 (Mobile First)
- **접근성**: WCAG AA 준수
- **이미지**: 모든 이미지에 `next/image` 사용
- **기본 속성**: 한국어 콘텐츠는 UTF-8 인코딩 및 `lang="ko"` 지정
