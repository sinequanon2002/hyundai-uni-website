## 1. 경쟁사 분석 요약 (업박스)

### 강점
- **즉시 견적** 플로우: 첫 화면에서 바로 폐기물 종류 선택 → 배출지 입력 → 즉시 견적 유도
- **명확한 CTA**: "지금 바로 견적받기" 1개 핵심 버튼에 집중
- **Pain Point 선제 제시**: "폐기물 처리, 어떻게 해야 할지 모르겠다면" 식의 공감형 헤드라인
- **신뢰 지표 노출**: 처리 완료 건수, 파트너사 수 등 수치 데이터 상단 배치
- **모바일 최적화**: 단계별 Wizard UI로 모바일 전환율 제고

### 약점 (우리 사이트에서 차별화 가능한 부분)
- 일반폐기물 중심 → **지정폐기물 전문성** 강조로 차별화
- 플랫폼(중개) 모델 → **직접 수거·운반 허가업체** 신뢰도 우위
- 법적 전문성 콘텐츠 부족 → **올바로시스템, 처리 증빙** 등 전문 콘텐츠 강점

## 2. 현재 랜딩페이지 문제점

| 항목 | 현재 상태 | 개선 방향 |
|------|----------|----------|
| 섹션 순서 | Hero → 서비스 → Why Us → 거래처 | Hero → 신뢰 수치 → Pain Point → 서비스 순으로 PAS 구조 강화 |
| 신뢰 수치 위치 | 거래처 섹션 내 묻혀 있음 | Hero 바로 아래 독립 섹션으로 상단 노출 |
| Pain Point 섹션 | 없음 | 고객 공감 유발 후 서비스 소개로 전환 유도 |
| CTA 전략 | 단일 CTA | 소프트 CTA(소개서/블로그) + 하드 CTA(견적 신청) 이중 전략 |
| 법적 리스크 강조 | 없음 | 완곡하고 공감적인 표현으로 필요성 환기 |

## 3. 개선된 랜딩페이지 섹션 구조 (PAS 프레임워크)

```
HeroSection          — 브랜드 첫인상 + 핵심 메시지
TrustStatsSection    — 수치로 즉각 신뢰 형성 (300+, 5,000t+, 99.8%, 24h)
PainPointSection     — 고객 Pain Point 공감 → 솔루션 연결
ServicesSection      — 서비스 상세 소개
WhyUsSection         — 차별화 이유
ClientsSection       — 거래처 사례
ProcessSection       — 견적~처리 프로세스
CtaSection           — 최종 전환 유도
```

## 4. Pain Point 섹션 콘텐츠 기획

### 설계 원칙
- **공감 우선**: "어렵다", "놓칠 수 있다" 등 부드러운 표현 사용
- **법적 리스크 완곡화**: "법적 문제가 생길 수 있습니다" → "놓치면 어려움이 생길 수 있습니다"
- **솔루션 제시**: 각 Pain Point마다 현대유앤아이의 해결책 병기

### 4개 카드 내용

| # | 아이콘 | 제목 | 핵심 메시지 |
|---|--------|------|------------|
| 1 | ClipboardList | 올바로시스템 전자인계서, 익숙하지 않으신가요? | 작성 대행 + 오류 없이 적법 처리 |
| 2 | BookOpen | 폐기물 종류마다 다른 보관 기준, 다 파악하기 어렵죠? | 현장 방문 시 무료 보관 기준 확인 |
| 3 | ShieldCheck | 위탁업체 자격 요건, 꼼꼼히 확인하고 계신가요? | 허가 서류 투명 제공 |
| 4 | FileCheck | 처리 이력과 증빙 서류, 체계적으로 관리되고 있나요? | 처리 확인서 + 전자인계서 사본 제공 |

### 브릿지 CTA 문구
> "전자인계서 대행부터 처리 이력 관리까지 — **현대유앤아이**가 모든 과정을 함께합니다."

## 5. TrustStatsSection 수치 데이터

| 수치 | 레이블 | 부제 |
|------|--------|------|
| 300+ | 누적 거래처 | 대구·경북 전 지역 |
| 5,000t+ | 연간 처리량 | 지정폐기물 전 품목 |
| 99.8% | 고객 재계약률 | 높은 서비스 만족도 |
| 24h | 견적 응답시간 | 신속 현장 방문 견적 |

- 배경: `bg-primary` (딥블루 #1F4E79)
- Hero 바로 아래 배치하여 스크롤 없이 신뢰 지표 노출

## 6. 추가 개선 권고 사항 (미구현)

- [ ] 올바로시스템 전용 랜딩 섹션 강화 (경쟁사 대비 명확한 차별점)
