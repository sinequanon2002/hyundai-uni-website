## 2. 업박스 웹사이트 UX·UI 분석

### 2-1. 랜딩페이지 구조 (상→하)

1. **Hero** — "폐기물, 지금 바로 처리하세요" 식의 즉각적 행동 유도 헤드라인
2. **즉시 견적 입력폼** — 첫 화면에서 폐기물 종류 + 배출지 선택으로 즉시 견적 개시
3. **신뢰 지표** — 처리 건수, 파트너사 수, 고객 평점 등 수치 배너
4. **이용 방법** — 3단계 또는 4단계 아이콘 플로우 (신청 → 매칭 → 수거 → 완료)
5. **서비스 카테고리** — 폐기물 종류별 아이콘 그리드
6. **고객 후기** — 실제 사용 후기 슬라이더
7. **파트너사** — 협력 업체 로고 배열

### 2-2. UX 강점

| 요소 | 업박스 방식 | 평가 |
|------|-----------|------|
| CTA 집중도 | 견적 신청 1개 버튼에 집중 | ★★★★★ |
| 첫 화면 전환 깔때기 | Hero에서 바로 입력폼 노출 | ★★★★★ |
| 모바일 최적화 | 단계별 Wizard UI, 터치 친화적 | ★★★★☆ |
| 신뢰 지표 노출 위치 | 상단 가시권에 수치 배치 | ★★★★☆ |
| 공감형 헤드라인 | Pain Point 언어 사용 | ★★★★☆ |
| 처리 프로세스 시각화 | 단순 3~4단계 아이콘 | ★★★☆☆ |

### 2-3. UX 약점

- **전문성 콘텐츠 부족**: 법적 요건, 처리 기준 등 심층 정보 없음
- **업종별 맞춤 콘텐츠 없음**: 모든 고객을 단일 페이지로 처리
- **중개 플랫폼의 신뢰 한계**: 실제 처리 업체가 누구인지 불투명
- **지정폐기물 미취급**: 허가·법적 요건이 복잡한 지정폐기물은 서비스 범위 외

## 3. 우리 사이트(현대유앤아이)와 비교

### 3-1. 포지셔닝 비교

| 구분 | 업박스 | 현대유앤아이 |
|------|-------|------------|
| 폐기물 범위 | 일반폐기물 | **지정폐기물 전문** |
| 사업 모델 | 중개 플랫폼 | **직접 수거·운반 허가업체** |
| 법적 허가 | 중개업 | **폐기물 수집·운반업 허가** |
| 지역 커버리지 | 전국 | 대구·경북 특화 |
| 올바로시스템 | 미지원 | **전자인계서 대행 포함** |
| 처리 증빙 | 기본 영수증 수준 | **처리 확인서 + 인계서 사본** |
| 타겟 고객 | B2C/소상공인 | **B2B (제조·의료·공공기관)** |

### 3-2. 우리의 명확한 우위

1. **법적 전문성**: 지정폐기물 수거·운반 허가 보유 → 배출자 법적 책임 분산
2. **올바로시스템 대행**: 전자인계서 작성 대행 서비스 → 경쟁사 대부분 미지원
3. **직접 처리**: 플랫폼 중개가 아닌 직접 출동 → 책임 소재 명확
4. **전문 거래처**: 대기업 협력사, 의료기관, 공공기관 등 고신뢰 레퍼런스
5. **처리 이력 관리**: 증빙 서류 체계적 제공 → 환경부 점검 대비 가능

### 3-3. 우리가 배워야 할 점

1. **즉시 전환 구조**: 첫 화면에서 바로 견적 신청 유도 (현재: 여러 클릭 필요)
2. **Pain Point 선제 공감**: "어떻게 처리해야 할지 모르겠다면" 식의 고객 언어 사용
3. **신뢰 수치 상단 노출**: 스크롤 없이 볼 수 있는 위치에 수치 배치
4. **프로세스 단순화**: 복잡한 프로세스를 3~4단계로 압축해서 시각화
5. **모바일 견적 UX**: 모바일에서 견적 신청 마찰을 줄이는 단계별 입력 흐름

## 4. 콘텐츠·메시지 전략 비교

### 4-1. 헤드라인 전략

| | 업박스 방식 | 현대유앤아이 방향 |
|--|-----------|----------------|
| 헤드라인 | 즉각 행동 유도 ("지금 바로") | **전문성 + 신뢰 강조** ("적법하게, 안심하고") |
| 서브 카피 | 편의성 강조 | **위험 회피 + 전문 솔루션** |
| CTA 문구 | "견적 받기" | "무료 방문 견적 신청" (전문성 + 무료 강조) |

### 4-2. 신뢰 구축 방식

| | 업박스 | 현대유앤아이 |
|--|-------|------------|
| 수치 | 플랫폼 거래량 | 누적 거래처, 재계약률, 연간 처리량 |
| 후기 | 일반 사용자 리뷰 | B2B 담당자 레퍼런스 (예정) |
| 허가 증빙 | 중개업 등록 | **환경부 수거·운반 허가증** |
| 전문성 | 없음 | 올바로시스템, 지정폐기물 법령 콘텐츠 |

## 7. 핵심 시사점 요약

> 업박스는 **편리함과 즉시성**으로 일반폐기물 시장을 공략한다.  
> 현대유앤아이는 **전문성과 법적 신뢰**로 지정폐기물 B2B 시장에서 차별화해야 한다.

- 업박스의 UX 구조(상단 노출, 즉시 전환)는 배워야 하지만
- 콘텐츠 방향은 반드시 **지정폐기물 전문 업체**로서의 법적·기술적 신뢰 중심으로 구성해야 한다.
- 고객(배출 사업장 담당자)의 핵심 Pain Point는 **법적 리스크 회피**와 **서류 부담 최소화**이므로
  이 두 가지를 해결해 주는 파트너로 포지셔닝하는 것이 장기적 경쟁 우위가 된다.
