## 개선 전략 방향

업박스 분석에서 도출한 핵심 인사이트는 두 가지다.

**① 현대유앤아이는 이미 가진 전문성을 충분히 보여주고 있지 않다**  
정식 허가 번호, 올바로시스템 대행, 처리 증빙 제공 — 이 세 가지는 경쟁사가 흉내 낼 수 없는 차별점인데, 현재 웹사이트에서 전면에 드러나지 않는다.

**② 방문자 100명 중 95명은 "지금 당장 견적을 낼 준비"가 안 됐다**  
업박스는 소개서 다운로드, 웨비나 등으로 이 95명을 리드로 확보한다. 현재 현대유앤아이는 5명만 잡고 95명을 보낸다.

```
   → 손실 회피 심리 자극, 고관심 키워드

③ "지정폐기물 위탁업체 선정 기준 — 배출자 연대책임 피하는 법"
   → Pain Point 직접 겨냥, 검색 의도 명확

④ "대구·경북 제조업체 지정폐기물 처리 비용, 현실적으로 알아보기"
   → 지역 SEO, 비용 키워드 유입
```

### 3-4. 현장갤러리 Before/After 개편

**현재**: 현장 사진 나열  
**개선**: Before/After 형식 + 업종 태그 필터

```
필터: 전체 | 제조업 | 의료기관 | 화학업 | 공공기관
각 카드: Before 사진 + After 사진 (슬라이더) + 업종 태그 + 처리 품목
```
