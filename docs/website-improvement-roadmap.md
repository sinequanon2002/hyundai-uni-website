## 핵심 전략 방향 (v3 업데이트)

### 업박스 분석에서 도출한 최우선 인사이트

#### 1. ESG가 새로운 전환 트리거다

2026년 ESG 공시 의무화 시행으로 중견기업 이상은 폐기물 처리 이력을 외부에 공시해야 한다.  
업박스는 이를 "UBIS 임팩트 스코어"로 전면 활용한다.  
현대유앤아이는 처리 증빙 서류 제공이라는 강점이 있으므로 ESG 포지셔닝을 즉시 활용 가능하다.

```
현재 메시지: "수거부터 올바로시스템 신고, 처리 증빙 서류까지 대행"
ESG 강화 후: "지정폐기물 처리 이력 → ESG 보고서 데이터로 즉시 활용"
```

#### 2. 지인추천이 가장 저비용 고신뢰 리드다

B2B 폐기물 업계에서 추천 기반 리드는 전환율이 3~5배 높다.  
업박스는 20~50만원 상품권으로 추천을 구조화했다.  
현대유앤아이는 기존 300+ 거래처가 있어 추천 풀이 충분하다.

#### 3. 케이스 스터디가 업종별 신뢰의 핵심이다

"우리 업종도 했다"는 사회적 증거가 "서비스 설명"보다 3배 강력하다.  
업박스는 호텔·쇼핑몰·물류 3개로 각 업종 담당자를 정확히 겨냥한다.  
현대유앤아이는 실제 처리 이력이 있으므로 콘텐츠 수집만 되면 즉시 구현 가능하다.

#### 4. 웨비나는 가장 강력한 B2B 리드 수집 채널이다

업박스 웨비나 특징:
- 40분 짜리 실무 콘텐츠 (가볍지 않다)
- 신청자에게 발표 자료 + 비용 계산기 제공
- 비용 시뮬레이터, 스타벅스/배달 기프트카드 증정
- 웨비나 등록 = 의향 높은 고객임을 바로 파악 가능

현대유앤아이는 "올바로시스템 입문" 웨비나 하나만 운영해도 강력한 차별화 가능.

#### 5. 품목 특화 페이지는 롱테일 SEO의 핵심이다

업박스 `/신규-폐식용유`는 단일 품목 페이지지만 "폐식용유 수거 업체" 검색자를 직접 전환한다.  
현대유앤아이의 80+ 품목 데이터를 개별 페이지로 만들면 롱테일 SEO 최적화 가능.

> 업박스 UBIS: 고객사별 처리량 → 탄소절감량 → ESG 리포트 데이터로 활용.  
> 현대유앤아이 버전: 내부 처리 데이터 집계 → 웹사이트 수치 + 고객 인증서 발급.

```
관리자에서 처리 데이터 입력
  └→ 메인 페이지 ImpactSection 수치 자동 업데이트
  └→ 고객용 "연간 처리 실적 인증서" PDF 자동 생성
  └→ 고객 마이페이지에서 다운로드 가능

구현:
  - Supabase impact_stats 테이블
  - /api/impact/summary Server Action
  - PDF 생성: @react-pdf/renderer
```
