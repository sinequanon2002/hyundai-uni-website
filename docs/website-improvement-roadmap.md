## 핵심 전략 방향

업박스 전체 분석에서 도출한 가장 중요한 인사이트 세 가지:

### 1. 리드 수집 채널이 0개다

업박스는 **5개의 독립 리드 수집 채널**을 운영한다.  
현대유앤아이는 "견적 문의 폼" 하나뿐이다.  
방문자 100명 중 95명은 지금 당장 견적을 낼 준비가 안 된 상태다. 이 95명이 전부 이탈하고 있다.

```
업박스 리드 채널 5개          현대유앤아이 리드 채널
────────────────────         ───────────────────
웨비나 신청           →       ❌
올바로 가이드북 다운로드 →       ❌
폐기물 전략 리포트    →       ❌
서비스 소개서 다운로드 →       ❌ (버튼만 있고 페이지 없음)
견적 문의 폼          →       ✅
```

### 2. 블로그와 웹사이트가 연결되지 않는다

업박스는 `blog.upbox.us`에서 월 4~8회 포스트를 발행하고,  
각 포스트 내에서 소개서·가이드북·견적 문의로 유도한다.  
현대유앤아이 네이버 블로그는 웹사이트와 단절되어 있다.  
블로그 방문자가 웹사이트로 유입되지 않고 그냥 이탈한다.

### 3. 업종별 페이지가 없다

업박스는 8개 업종별 특화 페이지로 **"나와 같은 상황의 회사"**를 즉시 보여준다.  
각 페이지는 독립적인 SEO 키워드를 갖는다 ("제조공장 폐기물 처리" 등).  
현대유앤아이는 이 모든 키워드에서 노출되지 않는다.

### 2-3. 소개서 다운로드 페이지 신설

> 근거: 업박스 `/서비스-소개서` 페이지 — 이메일 수집 후 소개서 발송. 현대유앤아이는 버튼만 있고 실제 페이지가 없다.

**구현 스펙:**
```

  ① 헤드라인: "현대유앤아이 서비스 소개서를 받아보세요"

  섹션 1: 보유 차량 (기종별 사진 + 용도 설명 + 대수)
  섹션 2: 수거 전용 설비 (탱크, 용기, 도구 등)
  섹션 3: 올바로시스템 연동 처리 프로세스
  섹션 4: 허가 증빙 문서 이미지
```

---

### 3-4. 전환 유도 전용 페이지 신설

> 근거: 업박스 `/바꾸면-바뀐다` — "이용중인 폐기물 업체 정말 좋은게 맞나요?" 라는 헤드라인으로  
> 경쟁사 사용 고객을 직접 전환 유도하는 별도 랜딩페이지.

**구현 스펙:**
```
URL: /switch
파일: src/app/(public)/switch/page.tsx

헤드라인: "지금 거래하는 업체, 정말 적법하게 처리하고 있나요?"
서브카피: "위탁업체 자격 미확인 시 배출자에게도 책임이 따릅니다"

구성:
  ① 업체 자가 점검 체크리스트 (5개 질문)
     - 수집·운반 허가 번호를 확인한 적 있나요?
     - 올바로시스템 전자인계서를 직접 작성하고 있나요?
     - 처리 완료 증빙 서류를 받고 있나요?
     - 보관 기준 위반 여부를 점검받은 적 있나요?
     - 24시간 내 견적을 받을 수 있나요?

  ② "현대유앤아이로 바꾸면 달라지는 것" (ComparisonSection 내용 재활용)

  ③ 전환 고객 후기 (있을 경우)

  ④ CTA: "지금 무료 방문 견적으로 전환 시작하기"
```

---

### 3-5. 현장갤러리 Before/After 개편

**현재**: 현장 사진 나열  
**개선**: Before/After 슬라이더 + 업종 태그 필터

```
필터 탭: 전체 | 제조업 | 의료기관 | 화학업 | 공공기관
각 카드: Before 사진 / After 사진 + 업종 태그 + 처리 품목 텍스트
```

### 4-1. 블로그 콘텐츠 로드맵

> 근거: 업박스는 블로그를 통해 "차가운 리드"를 웹사이트로 유입시키고 가이드북/견적으로 전환한다.  
> 현재 현대유앤아이 네이버 블로그는 웹사이트와 연결이 없어 유입 효과가 없다.

**포스트 제목 패턴 (업박스 학습):**
- `[무료 다운로드]`, `[체크리스트]` 대괄호 프리픽스 사용
- 구체적 수치 포함 ("1분이면", "과태료 얼마")
- 실무 오류·문제 상황 직접 언급 ("기한 초과 시", "오류 발생하면")

**우선 발행 포스트 12개 (분기별 3개씩):**

| # | 제목 | 카테고리 | SEO 키워드 |
|---|------|---------|-----------|
| 1 | [체크리스트] 올바로시스템 전자인계서 제출 전 확인할 10가지 | 올바로시스템 | 올바로시스템 사용법 |
| 2 | 올바로시스템 기한 초과 인계서, 이렇게 처리하세요 | 올바로시스템 | 올바로시스템 기한초과 |
| 3 | 지정폐기물 위탁업체 선정 기준 — 배출자 연대책임 피하는 법 | 법규 인사이트 | 지정폐기물 위탁업체 |
| 4 | 폐산 보관 기준 위반 시 과태료, 실제 사례로 알아보기 | 법규 인사이트 | 지정폐기물 보관기준 |
| 5 | [무료 다운로드] 지정폐기물 담당자 연간 관리 체크리스트 | 자료 배포 | 지정폐기물 관리 |
| 6 | 대구 제조업체 지정폐기물 처리 비용, 현실적으로 알아보기 | 비용 안내 | 지정폐기물 처리비용 |
| 7 | 폐유기용제 처리 방법 — 종류별 완전 정리 | 품목 가이드 | 폐유기용제 처리 |
| 8 | 의료폐기물 vs 지정폐기물, 다른 점이 뭔가요? | 품목 가이드 | 의료폐기물 |
| 9 | 올바로시스템 없이 폐기물 처리하면 어떻게 될까? | 법규 인사이트 | 올바로시스템 의무 |
| 10 | ESG 보고서에 지정폐기물 처리 실적을 활용하는 방법 | ESG 트렌드 | ESG 폐기물 |
| 11 | 현대유앤아이가 대구 지정폐기물 1위인 이유 | 브랜드 스토리 | 대구 지정폐기물 |
| 12 | [후기] 경산 제조업체 폐산 50드럼, 3일 만에 처리한 사례 | 고객 사례 | 폐산 수거 |

### 4-2. 소개서 PDF → 이메일 발송 자동화

Phase 2에서 직접 다운로드로 시작한 것을 이메일 수집 + 자동 발송으로 업그레이드:

```
폼 제출 → Supabase 저장 (이름, 회사, 이메일, 업종)
         → Resend로 소개서 PDF 이메일 자동 발송
         → 3일 후 후속 이메일 (가이드북 추천)
         → 7일 후 견적 문의 유도 이메일
```
